# Connect-the-Dots-Editor
This is an interactive tool based on Python (Processing / py5 style) that allows users to manually place points on an image to create "Connect the Dots" drawings.

Features:
Load a reference image
Manually place points using the mouse
Automatically record the sequence of points
Generate a numbered dot structure

Controls:
Left click: add a point
Right click: stop placing points

How to Run:
This project requires a Processing / py5-style environment.
Recommended setup:
Thonny
thonny-py5mode plugin
